# Java 面试宝典

> 📖 **面试不是背题，而是展示你对技术的理解和思考过程。** 本宝典以真实面试场景中的 **问-答** 形式组织，涵盖高频底层原理题和真实场景题，帮你建立系统化的面试知识体系。

## 使用方式

- **⭐️ 星级说明**：⭐️⭐️⭐️ 最高频必考 | ⭐️⭐️ 高频常见 | ⭐️ 进阶加分题
- 每道题包含**核心回答**（面试时可口述）和**深度扩展**（加分项），面试官追问和常见错误帮你避开陷阱
- 建议按分类顺序系统学习，也可利用右侧目录锚点快速定位薄弱知识点
- 答案中引用了学习笔记中的详细章节（如 `详见 [HashMap 深度解析](../collection/map.md)`），可交叉阅读加深理解
