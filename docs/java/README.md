建议按以下顺序学习：

1. **Java 基础** —— 掌握核心概念、基本语法、面向对象
2. **集合框架** —— 理解容器体系，深入 HashMap、ArrayList 源码
3. **并发编程** —— 线程池、锁机制、JUC 工具类
4. **JVM 原理** —— 内存结构、GC、类加载
5. **Java 新特性** —— Lambda/Stream、模块系统、虚拟线程
