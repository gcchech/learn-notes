# 关于本站

## 📝 简介

Java 学习笔记是一个以 **VuePress 2** + **VuePress Theme Hope 2** 构建的 Java 技术博客分享网站。

本站内容涵盖 Java SE 核心、集合框架、并发编程、JVM 调优、Java 新特性以及常用开发工具，力求为 Java 学习者提供**系统化、高质量**的技术文章。

## 🎯 内容特色

- **深度优先**：不止讲"是什么"，更讲"为什么"和底层原理（经常附带字节码级别的验证）
- **代码驱动**：每个概念都配有可运行的代码示例，拒绝纯理论说教
- **面试导向**：标注高频考点 ⭐️，提供记忆口诀和对比表格
- **持续更新**：跟进最新 Java 版本（当前覆盖至 JDK 21）
